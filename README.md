# Usage

<!-- usage -->

```sh-session
$ npm install -g storytoy-cli
$ storytoy COMMAND
running command...
$ storytoy (-v|--version|version)
storytoy-cli/0.1.0 linux-x64 node-v16.3.0
$ storytoy --help [COMMAND]
USAGE
  $ storytoy COMMAND
...
```
